Assumptions and notes

Rectangular and circular concrete columns. Metric inputs only. Longitudinal steel includes one lap per vertical bar using the uploaded lap table. External and internal ligs/hoops include two 135° fitment hooks per piece using Australian standard hook lengths. Designs are stored in the backend JSON data file and compiled into a schedule. Bundled vertical bars are limited to bundles of 2.

  • Longitudinal cut length = storey height + one lap per vertical bar from the uploaded lap table.
  • Each lig / hoop / crosstie includes two 135° fitment hooks using Australian standard hook lengths.
  • Hook lengths use the AS 3600 / Australian standard 135° fitment hook dimensions for the selected lig bar size.
  • Lig count along height = ceil(storey height / spacing) + 1 so both ends are included.
  • Rectangular total bar positions = 2(N + M) − 4. If bundled bars are selected, total longitudinal bar count = positions × 2.
  • Rectangular internal ligs per layer = (bars along width − 2) depth-spanning pieces + (bars along depth − 2) width-spanning pieces.
  • Circular columns use one hoop per lig level and no internal ligs. N10 is available for ligs with Australian standard 135° fitment hook lengths.
  • Lap lookup is classified as 40 MPa for 40–49 MPa, 50 MPa for 50–64 MPa, and ≥65 MPa for 65 MPa and above.
  • Bundled bars affect longitudinal steel quantity only and are limited to bundles of 2. Lig geometry still follows the entered bar positions.
  • Excel import reads the exported workbook template and recalculates each imported row before saving.
  • JSON import reads the exported JSON file, recalculates each imported row before saving, and merges by id when present.
